# week7-学习笔记

### trie
  基本结构
  字典树,又称单词查找树或键树,是一种树形结构.典型应用是用于统计和排序大量的字符串(但不仅限于字符串),所以经常被搜索引擎系统用于文本词频统计
  它的优点是:
  1.最大限度的减少无谓的字符串比较
  2.查询效率比哈希表快
